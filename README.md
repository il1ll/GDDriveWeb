# GDDrive
Cloud Storage Implemented in Geometry Dash

Video: https://youtu.be/oENqzFJ3TgI

# Installation

clone repo
```bash
git clone https://github.com/il1ll/GDDriveWeb.git
cd GDDriveWeb
```

install packages
```bash
pip install flask requests
```

run flask server
```bash
python gddrive.py
```

now enter [http://localhost:5000/](http://localhost:5000/)

and enjoy :D!
